# Site images (meta / tab)

- **Tab icon (favicon)**  
  Put your icon here as `favicon.png` (e.g. 32×32 or 48×48 px).  
  It will be used as the small icon in the browser tab (replaces the default globe).

- **Default preview image (Open Graph / social)**  
  Put your image here as `og-image.jpg` (recommended size **1200×630** px).  
  It is used when the site is shared (e.g. social, links) and in meta previews.  
  To use a different path, change `defaultImage` in `src/lib/seo.ts`.
